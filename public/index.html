About IAB Taxonomy Explorer

A free, open-source tool for exploring IAB Tech Lab taxonomies used in programmatic advertising. Supports Content Taxonomy (versions 1.0 through 3.1), Ad Product Taxonomy (versions 1.0 through 2.0), and Audience Taxonomy (versions 1.0 and 1.1).

Resolve OpenRTB cattax values (1–6) to full category objects, translate category IDs between taxonomy versions using official IAB mapping files, and search across all taxonomy versions simultaneously.
